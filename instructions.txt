Challenge Mobile App Eng

Instrucciones:
La duración estimada para completar el examen es de un máximo de 5 días.

App de recetas
Configure un nuevo proyecto de Flutter que incluya las siguientes dependencias:
● Dio para realizar solicitudes HTTP.
● Utilizar Flutter bloc para gestionar el estado de la aplicación.
● Utilizar la API de https://www.themealdb.com/api.php
Asegúrese de manejar posibles errores durante la solicitud a la API. En caso de un
error de red, muestre un mensaje de error en la interfaz de usuario.
Debe contener 3 secciones:
● Home (Lista de recetas paginada)
● Detalle de receta
● Buscador de recetas

Pruebas Unitarias

Escriba al menos una prueba unitaria para una función importante de su aplicación.
Asegúrese de que la prueba verifique el comportamiento correcto, incluso en
condiciones de error.
Plus:
Incluir UX y UI.
